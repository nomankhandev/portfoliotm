# 🚀 Noman Khan — Developer Portfolio

A modern, dark-themed personal portfolio website built with HTML, CSS, JavaScript, Bootstrap, jQuery, and deployed with Git.

---

## 📁 Project Structure

```
noman-portfolio/
│
├── index.html      → Main HTML file (all sections)
├── style.css       → Custom CSS (dark theme, animations)
├── script.js       → jQuery logic (typing, filter, scroll, form)
└── README.md       → This file
```

---

## 🛠️ Tech Stack

| Technology   | Purpose                         |
|--------------|---------------------------------|
| HTML5        | Structure & Semantic Markup     |
| CSS3         | Styling, Animations, Variables  |
| Bootstrap 5  | Grid, Navbar, Responsive Layout |

## ✨ Features

- 🌑 **Dark Modern Theme** with CSS custom properties

## 🚀 Setup in VS Code

### 1. Open in VS Code
```bash
# Clone or open folder
code noman-portfolio
```

### 2. Use Live Server Extension
- Install **Live Server** extension in VS Code
- Right-click `index.html` → **"Open with Live Server"**
- Portfolio opens at `http://127.0.0.1:5500`

## 🔧 Git Setup & Deployment

### Initialize Repository
```bash
git init
git add .
git commit -m "🚀 Initial commit — Noman Khan Portfolio"
```

### Push to GitHub
```bash
git remote add origin https://github.com/YOUR_USERNAME/noman-portfolio.git
git branch -M main
git push -u origin main
```

### Deploy on GitHub Pages
1. Go to your GitHub repository
2. **Settings** → **Pages**
3. Source: **Deploy from a branch** → `main` → `/(root)`
4. Your portfolio will be live at:
   `https://YOUR_USERNAME.github.io/noman-portfolio/`
